## 💡 О проекте

BobiVPN Checker — это мощный автоматизированный скрипт, предназначенный для парсинга и глубокой проверки тысяч VPN-ключей. В отличие от стандартных чекеров, которые просто пингуют TCP-порт, наш инструмент выполняет **полноценное подключение** и маршрутизацию трафика, имитируя реальное поведение VPN-клиента (например, v2rayNG или NekoBox) в режиме TUN.

### 🔥 Поддерживаемые протоколы
`VLESS` (Reality, TLS, WS) • `VMess` • `Trojan` • `Shadowsocks` • `Hysteria2`

## ⚙️ Как это работает (Двухэтапная проверка)

Мы используем ядро **Xray-core** напрямую внутри Go-приложения (in-process), что исключает накладные расходы на запуск внешних процессов.

### ⚡ Этап 1: Быстрый фильтр (ProxyGet)
Первичная проверка, способная обрабатывать **тысячи ключей одновременно**. 
* **Что делает:** Чекер поднимает локальный HTTP-клиент, проксирующий трафик через ядро Xray.
* **Цель:** Выполнить быстрый GET-запрос к `generate_204` (Cloudflare/Google).
* **Результат:** Моментально отсеиваются все мертвые, заблокированные или не отвечающие сервера.

### 🛡️ Этап 2: Глубокий тест (Deep Check)
Те ключи, которые выжили на первом этапе, подвергаются суровому тестированию.
* **IP-Проверка:** Чекер делает запрос к `cloudflare.com/cdn-cgi/trace` через прокси. Если выходной IP-адрес совпадает с вашим реальным — значит ключ "течет" и помечается как нерабочий.
* **Геолокация:** Определяется реальная страна и провайдер (ISP) выходной ноды.
* **Тест скорости:** Загружается тестовый файл для определения пропускной способности (KB/s).
* **Итог:** 100% гарантия, что ключ работает так же, как если бы вы включили его на телефоне.

## 🚀 Использование (через GitHub Actions)

Самый простой и правильный способ использования — автоматизация через GitHub Actions. Скрипт будет сам запускаться по расписанию, проверять подписки и обновлять файлы в репозитории.

1. **Добавьте ваши подписки**
   * Запишите ссылки в файл `subscriptions.txt` (по одной на строку) 
   * **ИЛИ** добавьте их в GitHub Secrets: `Settings -> Secrets and variables -> Actions -> New repository secret` (Название: `SUBSCRIPTION_URLS`).
2. **Загрузите код в репозиторий**
   Просто сделайте `git push` этого кода в свой репозиторий.
3. **Наслаждайтесь**
   Action автоматически скомпилирует Go-код, запустит проверку и обновит файлы.

## 💻 Локальный запуск (Терминал)

Если вы хотите запустить чекер на своем ПК (Windows, Linux, macOS).

```bash
# 1. Скачайте зависимости
go mod tidy

# 2. Соберите бинарный файл
go build -o scripts/vpn_checker_bin .

# 3. Запустите проверку
# Запуск с указанием параллельности
sudo ./scripts/vpn_checker_bin --concurrency-stage1=500 --concurrency-stage2=50
```

## Как запустить локально (Windows - только Этап 1)

Если вы запускаете на Windows, вы можете пропустить второй (TUN) этап:
```powershell
.\scripts\vpn_checker_bin.exe --concurrency-stage1=500 --skip-stage2
```

## 📁 Форматы вывода

После завершения работы чекер генерирует полностью готовые файлы для распространения:

| Файл | Описание |
| :--- | :--- |
| 📄 `vpn.txt` / `vpn_base64.txt` | Оригинальные ключи, прошедшие проверку. |
| ✨ `vpn_renamed.txt` | Ключи с красивыми именами (флаг страны + название провайдера). |
| 🦊 `bobi_vpn.txt` | Готовый профиль подписки для приложения **Happ** с кастомными заголовками. |
| ⚡ `bobi_vpn_lite.txt` | Урезанная версия подписки, содержащая только топовые страны (RU, DE, FR, FI, EE, LV, LT). |
| 📊 `vpn_report.json` | Подробная статистика (скорость, задержка, IP) по каждому ключу. |
| 🌍 `countries/` | Папка с отдельными файлами подписок, разбитыми по странам. |
